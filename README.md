# QRDrop

Generate QR codes for text, URLs, WiFi credentials, and vCards — entirely in your browser.

## Privacy

QRDrop runs 100% client-side. Your data never leaves your browser. No analytics, no tracking, no server calls.

## Features

- **Text** — Convert any plain text into a QR code
- **URLs** — Paste a link and get a scannable QR code
- **WiFi** — Share WiFi credentials (SSID, password, encryption type)
- **vCard** — Encode contact information (name, phone, email)
- **Download** as PNG or SVG
- **Copy** QR image to clipboard
- **Dark mode** with system detection and manual toggle
- **Multilingual** — EN, ES, FR, IT, PT, DE, CS
- **PWA** — Install as an app on any device
- **Live preview** — QR updates as you type

## Getting started

```bash
npm install
npm run dev
```
